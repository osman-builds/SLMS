# Student Learning Management System

Welcome to the **Student Learning Management System**, a console-based Java application for managing university data! 

## 📦 Features

This project is separated into two main modules:

### 1. Course Manager (`CourseManager.java`)
- **Add a Course**: Create a new course with a name, code, credit hours, summary, and MS Teams link.
- **Display Courses**: View all available courses formatted cleanly.
- **Search a Course**: Look up a specific course by its unique code.
- **Edit a Course**: Modify existing course details.
- **Delete a Course**: Remove a course from the catalog safely.

### 2. Student Manager (`StudentManager.java`)
- **Add a Student**: Save student details (Name, ID, Email, Phone Number).
- **Display Students**: View all registered students.
- **Delete a Student**: Remove a student safely.
- *(Search and Edit capabilities actively in development!)*

## 🚀 How to Run

1. Make sure you have **Java 17** (or compatible) and **Maven** installed.
2. Open the project in your favorite IDE (VS Code, IntelliJ, Eclipse, NetBeans).
3. Run the `main` method located in either of the core application files:
   - `src/main/java/com/mycompany/course/CourseManager.java`
   - `src/main/java/com/mycompany/course/StudentManager.java`
